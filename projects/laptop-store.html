Laptop Store

This is a laptop catalog website, using Node.js runtime environment and using a JSON file for storing and pulling data from.

The main page consists of a number of personal computer products, we can choose from. Here we can see, picture of the product, name, important information like screen size, CPU, and the price. Also, there is a link button called "Check it out" for a richer information page where we can also buy the product.

On the project information page, we see, a button "back" which will move us back to the main page where we see the products catalog, a product price, product name, and main product picture.

Below the same page, there is all the important technical information, in short, screen size, CPU model, RAM size, storage size, and weight. Below it is the same but richer information with a source link, and a buy button with a price on it. The click on the button will move you to the original page from which we took the information where we can buy the product that was showed.

Looking forward to adding a secure payment option of PayPal.
